# Побег из концлагеря

Это Telegram-бот, который представляет собой интерактивный текстовый квест. Игрок должен пройти регистрацию и двигаться по сюжету, делая выбор в ключевых точках повествования. Прогресс пользователя сохраняется между сеансами общения с ботом.

Игрок - советский солдат, который пытается сбежать из немецкого лагеря для военнопленных. Сюжет линейный с одним ключевым выбором, влияющим на концовку.

@mipt_quest_bot_123_bot

# Как запустить бота?

После клонирования репозитория, нужно определить переменную BOT_TOKEN в app.py.

Далее бот запускается следующим образом:

```bash
python -m venv .venv
./.venv/Scripts/activate
pip install -r requirements.txt
python app.py
```
